Iterates over all channels and installs a hook that will be called when the images in that channel are exported and imported.

This can be used to do fun things like post export tex makes etc.

managedAddVersion ()

source code  

This is an example of a managed add geo version.

It loads a geometry object from file, tagging our asset IDs onto it. It shows that each version can be managed individually with metadata tagged onto objects.
