Class RepeatType

source code

These are the types of behavior available for when the end of the frame range is reached.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  LOOP = 0
Return to the start of the frame range and continue playing.
  BOUNCE = 1
Play in reverse back to the start of the frame range, and then resume forward playback.
  STOP = 2
Stop playback.
