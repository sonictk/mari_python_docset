Class ColorspaceType

source code

These are used to define the data types used by colorspaces.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  COLORSPACE_TYPE_INT8 = 0
8 bit integer.
  COLORSPACE_TYPE_INT16 = 1
Integer more than 8 bits (not half float).
  COLORSPACE_TYPE_FLOAT = 2
Floating-point (including half float and double).
  COLORSPACE_TYPE_COUNT = 3
Total number of colorspace data types.
