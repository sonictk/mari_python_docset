resetAdjustmentLevelClampToDefault ()

source code  

In 2.6.v3, we changed the default value of clamping on the adjustment level filter to Clamp. This function parses through the objects->channels->layers to search for any level adjustment filter with a clamping value and reset it to its default 'Disable'.
