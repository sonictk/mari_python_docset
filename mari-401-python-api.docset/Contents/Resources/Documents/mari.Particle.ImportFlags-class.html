Class ImportFlags

source code

Specifies options for use in importing.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  IMPORT_CENTER_POSITIONS = 1
Subtracts the object's center position from each of the particle positions.
  IMPORT_BLEED_EDGES = 2
Turns on "full texture bleed" (full patch texture bleed is generated).
  IMPORT_FLATTEN = 4
The source layers/channel is flattened into a single image set prior to transfer ( Layer & channel transfer ONLY).
  IMPORT_IGNORE_HIDDEN = 8
The operation ignores hidden parts of the geometry.
  IMPORT_IGNORE_UNSELECTED = 16
The operation ignores unselected parts of the geometry.
