Class EventCoordinateSpace

source code

This enum defines which coordinate space the position of an event is in.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  CANVAS_SPACE = 0
Origin is at the top-left pixel of the currently active canvas.
  APPLICATION_SPACE = 1
Origin is at the top-left pixel of the main window.
  DESKTOP_SPACE = 2
Origin is at the top-left pixel of the desktop.
