Class PostFilterCollectionFlags

source code

Specifies the options that can be set on collections.

These options can be combined using binary 'or' operations - for example:

import mari

>>> filter_collection = mari.gl_render.createPostFilterCollection("New Filter Collection")

>>> filter_collection.setFlags(filter_collection.POSTFILTERCOLLECTION_EDITABLE | filter_collection.POSTFILTERCOLLECTION_REMOVABLE)

>>> mari.gl_render.deletePostFilterCollection(filter_collection)

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  POSTFILTERCOLLECTION_EDITABLE = 1
The collection can have filters added and removed by users.
  POSTFILTERCOLLECTION_REMOVABLE = 2
The collection can be removed by users.
  POSTFILTERCOLLECTION_RENAMABLE = 4
The collection can be renamed by users.
