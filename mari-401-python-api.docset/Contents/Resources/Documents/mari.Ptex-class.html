hash ( self )

source code  

Returns a unique identifier for this Ptex image set.

Mari maintains hashes of ptex data for internal book-keeping. These hashes are assumed to be universally unique (256bit skein hash). These hashes can be used to determine if an Ptex image set has changed since some given checkpoint. A different hash means different ptex data.

Returns: str
A hash string that uniquely identifies this version of this Ptex image set, or an empty string on failure.
