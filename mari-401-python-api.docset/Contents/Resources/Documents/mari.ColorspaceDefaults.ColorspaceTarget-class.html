Class ColorspaceTarget

source code

These are used to define the default colorspace of a data type or device.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  COLORSPACE_TARGET_INT8 = 0
Default for channels and, reading and writing image files containing 8 bit integer data.
  COLORSPACE_TARGET_INT16 = 1
Default for reading image files containing integer data more than 8 bits (not half float).
  COLORSPACE_TARGET_SCALAR8 = 2
Default for masks, heights, normals, depths, and other non-color images containing 8 bit data.
  COLORSPACE_TARGET_FLOAT = 3
Default for channels and, reading and writing image files containing floating-point data.
  COLORSPACE_TARGET_WORKING = 4
Default for painting, lighting, applying filters, and other such operations.
  COLORSPACE_TARGET_MONITOR = 5
Default for thumbnails.
  COLORSPACE_TARGET_COLOR_PICKER = 6
Default for color pickers and swatches.
  COLORSPACE_TARGET_BLENDING = 7
Default for applying blending operations such as, comp of the paint buffer and the current paint target, blending between layers, and combining inputs in a merge node.
  COLORSPACE_TARGET_COUNT = 8
Total number of colorspace targets.
Class Variable Details

COLORSPACE_TARGET_WORKING

Default for painting, lighting, applying filters, and other such operations. You will typically want to select the \"Linear\" colorspace to get accurate results.

Value: 
4

COLORSPACE_TARGET_MONITOR

Default for thumbnails. You will typically want to select the same colorspace as the viewer.

Value: 
5

COLORSPACE_TARGET_COLOR_PICKER

Default for color pickers and swatches. You will typically want to select the same colorspace as the current channel.

Value: 
6

COLORSPACE_TARGET_BLENDING

Default for applying blending operations such as, comp of the paint buffer and the current paint target, blending between layers, and combining inputs in a merge node. You will typically want to select the \"Default\" colorspace to get the same results as painting prior to Mari 3.0, and to get less banding artifacts when painting 8-bit channels and images.

Value: 
7
