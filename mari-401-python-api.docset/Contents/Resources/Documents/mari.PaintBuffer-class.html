                PaintBuffer

The paint buffer is the screen aligned plane that users paint onto.

It can be accessed from the CanvasManager - for example: mari.canvases.paintBuffer()

Example Code 

>>> # This example shows how to set or get the paint buffer depth
>>> import mari
>>> paint_buffer = mari.canvases.paintBuffer()
>>> original_depth = paint_buffer.depth()
>>> paint_buffer.setDepth(8)
>>> paint_buffer.depth()
8
>>> paint_buffer.setDepth(original_depth)

getPaint ( self )

source code  

Returns a QImage containing the current paint buffer.

This captures the paint in the paint buffer and returns it as an 8 bit RGBA QImage.

QImage does not support higher bit depths, so any float or half data will be reduced to 8 bits.

Returns: QImage
A QImage which is the same size as the paint buffer resolution, and contains 8 bit RGBA data

pickColor ( self , x , y )

source code  

Pick a color from the paintbuffer.

This allows a script to pick pixel color values from the paintbuffer. The x and y position must be in the range returned by resolution() The color returned may be a HDR value with r,g,b values above 1.0 This method is relatively slow to execute and should not be used to the query lots of pixels

#Pick a pixel in the middle of the screen

Size = mari.canvases.paintBuffer().resolution();

c = mari.canvases.paintBuffer().pickColor(Size.width()/2, Size.height()/2)

print c.toString()

Parameters:
  • x (int) - The x coordinate of the pixel.These coordinates are relative to the top left corner of the canvas
  • y (int) - The y coordinate of the pixel.These coordinates are relative to the top left corner of the canvas
Returns: Color
The color of the pixel at the paintbuffer location ( x , y )
Raises:
  • ValueError - Emitted if x or y are outside of the range (0,0)-resolution()

setPaint ( self , PaintImage )

source code  

Sets the contents of the paint buffer with a QImage.

This takes an RGBA QImage and copies it into the paint buffer. The QImage must be RGBA and the same resolution as the paint buffer (see resolution() ).

8 bit images will be converted to 16 or 32 bit where appropriate.

Parameters:
  • PaintImage (QImage) - The image to load into the paint buffer
Returns: bool
True if the paint was updated from the given image, or False otherwise
Raises:
  • ValueError - Raised if the image was invalid, or not of the correct resolution.

Attention: Any existing paint will be replaced without warning, so please be careful.
