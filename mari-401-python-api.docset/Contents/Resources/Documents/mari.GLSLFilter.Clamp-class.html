Class Clamp

source code

The strategy for handling out-of-range errors.

An out of range error occurs when a value is below 0 or above 1.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  CLAMP_TO_EDGE = 0
Repeat the edge pixel for out of range values.
  CLAMP_REPEAT = 1
Loop around in texture space, so -0.2 becomes 0.8, 1.2 becomes 0.2, and so on.
