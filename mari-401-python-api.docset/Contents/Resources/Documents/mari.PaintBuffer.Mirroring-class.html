Class Mirroring

source code

This enum controls mirroring in the paint buffer.

If enable mirroring repeats the users paint strokes in various different ways in the paint buffer. The default is MIRROR_NONE meaning only the strokes defined by the user are created. Mirroring is useful for painting symmetrical objects.

Note: These values are exposed in the parent class, but are also documented here for convenience.
