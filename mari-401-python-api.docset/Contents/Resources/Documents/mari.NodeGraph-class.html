            NodeGraph

Provides shader node graph functionality.

Example Code

This example creates a Perlin node and Merge node, connect them and sets a property 

>>> # obtain the root level node graph of the current geo entity
>>> import mari
>>> nodeGraph = mari.geo.current().nodeGraph()
>>>
>>> # Obtain the list of node types
>>> nodeTypeList = nodeGraph.typeList()
>>>
>>> # Create perlin noise node
>>> perlin = nodeGraph.createNode("Procedural/Noise/Perlin")
>>>
>>> # Create a merge node
>>> merge = nodeGraph.createNode("Layer/Merge")
>>>
>>> # Connect the perlin node to the Base input of the merge node
>>> merge.setInputNode("Base",perlin)
>>>
>>> # Obtain parameters of the merge node
>>> parameterNames = merge.metadataNames()
>>>
>>> # Set the BlendAmount of merge node to 0.5
>>> merge.setMetadata("Amount",0.5)

This example shows how to create a PaintNode 

>>> paintNode = nodeGraph.createPaintNode( 1024, 1024, 8, mari.Color(1.0,1.0,0.0,1.0) )

exportNodesAsArchive ( self , DirPath , Nodes , NoDataBlockCopy = False , UVIndexList = [ ] )

source code  

Saves the archive of the given nodes to the given location.

This saves the given nodes in an XML format and saves all dependencies into the given directory If the given location does not exist, it is created. If there are any exising files and directories, they will be deleted.

The nodes saved in the XML file as part of the archive are assigned with new unique UUIDs for sanity. The map returned by this function contains the mapping of the original node UUIDs to the unique UUIDs of the nodes in the XML file. A partial list of the new unique UUIDs can be given as an optional argument to importNodesFromArchive() to load a partial set of nodes from the archive.

Parameters:
  • DirPath (str) - The directory path to save the nodes archive to.
  • Nodes (list of Node ) - The list of nodes to save as archive.
  • NoDataBlockCopy (bool) - Whether to skip copying data blocks. If True, the images are copied only as template, but without texture data. This is False by default.
  • UVIndexList (list of int) - Pass in a list of zero-based UV indices if desired to specify the patches to export images for. If not specified or empty, images will be exported for all indices. The UDIM for a patch is the zero-based UV index plus 1001.
Returns: QHash of str, str
Returns the map of original node UUIDs to the new unique UUIDs stored in the node XML file of the archive.
