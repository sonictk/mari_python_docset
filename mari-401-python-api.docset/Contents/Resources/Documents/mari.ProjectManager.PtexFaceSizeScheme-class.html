Class Variable Details

PTEX_UNIFORM_SIZE

All faces all have a uniform texture size. The size is specified in the value of the PtexFaceSize meta option.

Value: 
0

PTEX_WORLD_SPACE_DENSITY_SIZE

Faces have texture sizes proportional to their model sizes. The texel density is specified in the value of the PtexFaceSize meta option.

Value: 
1
