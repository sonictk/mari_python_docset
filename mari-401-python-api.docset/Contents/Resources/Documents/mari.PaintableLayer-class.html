Paintable layers contain an image for each patch that can be painted directly.

You can access all types of layers from their containing Channel objects - for example: channel.layerList()

You can check for PaintableLayer objects by calling Layer.isPaintableLayer() : 

>>> # This examples creates a paintable layer and checks if it's a paintable layer
>>> import mari
>>> channel = mari.geo.current().currentChannel()
>>> layer = channel.createPaintableLayer("Test")
>>> layer.isPaintableLayer()
True

exportImages ( self , PathAndTemplate , Options = 0 , UVIndexList = [ ] )

source code  

Exports a set of images from the layer.

This is the same as selecting "Export Selected Layers" from the GUI with only the given layer selected. To export the entire layer stack, call LayerStack.exportImages() .

Pass in the path and template to export the files to - for example: 

>>> import mari
>>> layer = mari.geo.current().currentChannel().currentLayer()
>>> layer.exportImages('/tmp/$CHANNEL.$LAYER.$UDIM.tif')

Only paintable and procedural layers can be exported individually in this way.

Parameters:
  • PathAndTemplate (str) - The target path for the export and the template to export the files to, in a single string.
  • Options (int) - Pass in a value other than the default to turn on an option, such as disabling "small uniforms". See Image.SaveOptions .
  • UVIndexList (list of int) - Pass in a list of zero-based UV indices if desired to specify the patches to export images for. If not specified or empty, images will be exported for all indices. The UDIM for a patch is the zero-based UV index plus 1001.
Returns: None
Raises:
  • ValueError - Raised if the path and template parameter was empty, or if an invalid UV index was specified.
  • IOError - Raised if an error occurred with the export process.

See Also: ProceduralLayer.exportImages() , LayerStack.exportImages() , importImages()

importImages ( self , PathAndTemplate , ScaleOption = 1 , UVIndexList = [ ] , RemoveAlpha = False )

source code  

Imports a set of images into the layer.

This is the same as selecting "Import into Layer" from the GUI. To import into a layer stack, call LayerStack.importImages() .

Pass in the path to the files and the template to import them from - for example: 

>>> layer.importImages('/tmp/$CHANNEL.$LAYER.$UDIM.tif')
Parameters:
  • PathAndTemplate (str) - The path to the files and the template to import them from, in a single string
  • ScaleOption ( ImageSet.ScaleChoice ) - When the existing patch and the image in the file to import are of different sizes, this indicates which of the two to rescale to match the other
  • UVIndexList (list of int) - Pass in a list of zero-based UV indices if desired to specify the patches to export images for. If not specified or empty, images will be exported for all indices. The UDIM for a patch is the zero-based UV index plus 1001.
  • RemoveAlpha (bool) - Remove the alpha channel (convert RGBA to RGB).
Returns: None
Raises:
  • ValueError - Raised if the path and template parameter was empty, or if an invalid UV index was specified.
  • IOError - Raised if an error occurred with the import process.

See Also: ProceduralLayer.exportImages() , LayerStack.importImages() , exportImages()
