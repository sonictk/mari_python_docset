Class MeshOptions

source code

Mesh options for choosing the texturing method in projects.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  UV_OR_PTEX = 0
Use UVs if they exist, or Ptex if there are none; this is the default value.
  FORCE_PTEX = 1
Force Ptex mode.
