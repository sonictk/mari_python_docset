Class Channel

source code 

object --+        
         |        
  Metadata --+    
             |    
    LayerStack --+

                Channel

Channels are collections of layers that combine sets of image data to display using Shader objects.

You can access channels through their parent GeoEntity objects - for example: mari.geo.current().currentChannel()

Example Code 

>>> # This example shows how to obtain the current channel of the current GeoEntity object
>>> import mari
>>> channel = mari.geo.current().currentChannel()
