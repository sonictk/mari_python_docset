setFileTemplate ( self , rTemplate )

source code  

Sets the template used to generate the file name.

File templates are used to create the file names for each individual patch to load for the channel. The file names are copied from the template, with the following supported tokens replaced with their appropriate values:

  • $PATH - The base path for the textures, as returned by the path() function
  • $ENTITY - The name of the geometry object (or entity) that the images are being imported into
  • $CHANNEL - The name of the channel in the geometry object (or entity)
  • $LAYER - The name of the individual layer in the channel
  • $IMAGESET - The UUID of the channel in the layer
  • $UDIM - A special number indicating the index of the patch in the channel
  • $NUMBER - The number of the patch within the batch, starting from zero
  • $INDEX - Same as $NUMBER
  • $COUNT - The number of patches in the channel

See GeoPatch.udim() for a description of UDIM values.

Parameters:
  • rTemplate (str) - The template string to use for the patch file names
Returns: None

See Also: fileTemplate() , mari.examples.create_blacksmith_project.createBlacksmithProject()

__init__ ( self , rName , Width = 4096 , Height = 4096 , BitDepth = 8 , Scalar = False , pFillColor = None , FileSpace = 0 , ColorspaceSettings = None , ScalarColorspaceSettings = None )
(Constructor)

source code  

Constructor for ChannelInfo .

This class contains the information for creating a new channel. Note that this API is not complete, as not all options are fully supported yet.

Parameters:
  • rName (str) - Name of the channel. A value exception is raised if this is empty.
  • Width (int) - The width of the channel's textures in pixels. The default is 4K (4096).
  • Height (int) - The height of the channel's textures in pixels. The default is 4K (4096).
  • BitDepth ( Image.Depth ) - The bit depth of the channel's textures: mari.Image.DEPTH_BYTE (8, the default), mari.Image.DEPTH_HALF (16) or mari.Image.DEPTH_FLOAT (32).
  • Scalar (bool) - True if the channel should be scalar, or False (the default) for color.
  • pFillColor ( Color ) - Specifies a color to fill the channel's texture with on creation. If invalid, it uses a default gray.
  • FileSpace ( Image.FileSpace ) - The file space the channel will import and export data as: mari.Image.FILESPACE_NORMAL, mari.Image.VECTOR or mari.Image.FILESPACE_VECTOR_Y_FLIP.
  • ColorspaceSettings ( ColorspaceConfig ) - The colorspace settings for color data of the channel.
  • ScalarColorspaceSettings ( ColorspaceConfig ) - The colorspace settings for scalar data of the channel.
Returns: ChannelInfo
Raises:
  • ValueError - Raised if the channel name was empty, or the requested dimensions are not supported.
  • ValueError - Raised if invalid colorspace configurations were supplied.
Overrides: object.__init__

Deprecated: The Scalar parameter is no longer used, and will be removed in a future version.
