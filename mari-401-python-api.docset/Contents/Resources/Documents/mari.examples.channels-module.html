linkCurrentChannels ()

source code  

Sets up Mari to link current channel changes on all objects.

For example, if you have two objects with the same channel names, changing the channel on one of them will change to the same channel on the other.

The signal is only triggered when the channel actually changes, so in this case there is no danger of an infinite loop of channels continuously changing.
