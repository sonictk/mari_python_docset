Class InterpolationMethod

source code

The method of interpolation to use when calculating the final value from the closest particles on import.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  INTERPOLATION_NEAREST_NEIGHBOR = 0
The value of the closest particle.
  INTERPOLATION_INVERSE_DISTANCE_WEIGHTED = 1
The final value is the weighted average of the closest particles.
  INTERPOLATION_INVERSE_DISTANCE_WEIGHTED_SMOOTHER = 2
The final value is the weighted average, with a smoother falloff, of the closest particles.
  INTERPOLATION_INVERSE_DISTANCE_WEIGHTED_SHARPER = 3
The final value is the weighted average, with a sharper falloff, of the closest particles.
