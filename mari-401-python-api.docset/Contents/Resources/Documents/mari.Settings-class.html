Class Settings

source code 

object --+
         |
        Settings

A group of settings.

Users can load and save their own custom settings within Mari.

Example Code 

>>> # This examples shows how to store a user settings
>>> import mari
>>> settings = mari.Settings()
>>> settings.setValue("Test Setting","Hello World")
>>> print settings.value("Test Setting")
Hello World

beginGroup ( self , Prefix )

source code  

Appends prefix to the current group.

Starts a named nested set of values. All values set after this call and before the next endGroup() will be accessable only via the groups namespace.

Parameters:
  • Prefix (str) - Value to append
Returns: None

value ( self , Key , DefaultValue = None )

source code  

Returns the value for setting key.

If the setting doesn't exist, returns defaultValue. If no default value is specified, 'None' is returned.

If the key was set within a beginGroup() and endGroup() the group's path must be appended to the Key path

settings = mari.Settings()

settings.beginGroup('foo')

settings.setValue('bar', True)

settings.endGroup()

print settings.value('foo/bar')

Parameters:
  • Key (str) - Setting key
  • DefaultValue (variant) - Value returned if the setting doesn't exist
Returns: variant
The value of the setting, or the default if the setting doesn't exist.

See Also: setValue()
