Class SaveOptions

source code

Specifies optional behavior for image saving.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  DEFAULT_OPTIONS = 0
None of the other options are set.
  DISABLE_SMALL_UNIFORMS = 1
Turns off "small uniforms" (exporting blank textures as 8x8).
  ENABLE_FULL_PATCH_BLEED = 2
Turns on "full texture bleed" (full patch texture bleed is generated).
  REMOVE_ALPHA = 4
Remove the alpha channel (convert RGBA to RGB).
  BACKGROUND_EXPORT = 8
When set saveAs will return immediately and exporting will occur as a background job.
Class Variable Details

ENABLE_FULL_PATCH_BLEED

Turns on "full texture bleed" (full patch texture bleed is generated). This takes effect only for exporting textures.

Value: 
2

BACKGROUND_EXPORT

When set saveAs will return immediately and exporting will occur as a background job. Background jobs may not report errors or raise exceptions.

Value: 
8
