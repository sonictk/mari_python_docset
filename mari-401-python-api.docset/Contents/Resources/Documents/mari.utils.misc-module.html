message ( text , title = ' Message ' , buttons = 0 , icon = 0 , details = None )

source code  

Displays a simple Qt message box, with no return value.

This is a helper for the standard Qt message box, which is called using the function .exec(). Unfortunately "exec" is a keyword in Python, so we need to use some awkward syntax to get around it.

Parameters:
  • title - The title (or caption) of the message box
  • buttons - The buttons to show (the default is just the "OK" button)
  • icon - The icon to display on the box (the default is the 'i', or information, icon)
  • details - If specified, this provides extra text to display in a "Show Details" box
Returns:
None

Note: The parameters are in a different order to the standard Qt call. This is for convenience for users who just want to specify a message.

See Also: messageResult

getSaveFileName ( parent = None , caption = ' ' , dir = ' ' , filter = ' ' , selected_filter = None , options = 0 , save_filename = ' ' )

source code  

Returns a file name to save to, selected by the user.

If both dir and save_filename are supplied, the function combines the two if possible: an absolute path in save_filename will take precedence, but a relative path in save_filename will be appended to dir. Any file name component from save_filename will be appended to the resulting path.

Parameters:
  • parent - Parent widget to center the dialog over
  • caption - Title of the dialog box
  • dir - Initial working directory of the dialog. If it includes a file, the file will be selected
  • filter - Only files that match the given filter are shown
  • selected_filter - Initial selected filter
  • options - Qt option flags to indicate how to run the dialog (see this page for details)
  • save_filename - The default file name to save as. This may include a path if desired
Returns:
The name of the file selected by the user, or an empty string if the user pressed Cancel

setTimer ( interval_sec , func , single_shot = True )

source code  

Sets a once-off or repeating timer to call the given Python function.

Note: you must store the returned object, or it will be deleted and the timer function will not be called.

Parameters:
  • interval_sec (int) - Interval before calling the timer function, in seconds
  • func (callable) - The callable (e.g. function) to call after the specified time
  • single_shot (bool) - True to call the timer function once, or False to call repeatedly
