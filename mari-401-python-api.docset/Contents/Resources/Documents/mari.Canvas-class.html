Class Canvas

source code 

object --+
         |
        Canvas

A canvas is a Mari viewport.

Canvases are the geometry viewport containers in the Mari window.

You can access canvas objects through the CanvasManager - for example: mari.canvases.current()

Example Code: 

>>> # This example shows how to access the current canvas object and to print canvas size to the application log
>>> import mari
>>> canvas = mari.canvases.current()
>>> mari.app.log(str(canvas.size()))

Method Details

averageColor ( self , X , Y , Width , Height , Space = 1 )

source code  

Calculate the average color for an area in the canvas.

This allows a script to calculate the average pixel color values from the canvas. The x and y position must be in the range returned by size() The color returned may be an HDR value with r, g, b values above 1.0 This method is relatively slow to execute and should not be used to the query lots of pixels

#Pick a pixel in the middle of the screen

Size = mari.canvases.current().size();

c = mari.canvases.current().pickColor(Size.width()/2, Size.height()/2)

print c.toString()

Parameters:
  • X (int) - The x coordinate of the sample area. These coordinates are relative to the top left corner of the canvas
  • Y (int) - The y coordinate of the sample area. These coordinates are relative to the top left corner of the canvas
  • Width (int) - The width of the sample area. These coordinates are relative to the top left corner of the canvas
  • Height (int) - The height of the sample area. These coordinates are relative to the top left corner of the canvas
  • Space ( PickSpace ) - The space in which to pick pixels in
Returns: Color
The average color of the pixels at the canvas location (x, y, x + width, y + height)
Raises:
  • ValueError - Emitted if the sample area is outside of the range (0, 0) - size()

pickColor ( self , X , Y , Space = 1 )

source code  

Pick a color from the canvas.

This allows a script to pick pixel color values from the canvas. The x and y position must be in the range returned by size() The color returned may be an HDR value with r, g, b values above 1.0 This method is relatively slow to execute and should not be used to the query lots of pixels

#Pick a pixel in the middle of the screen

Size = mari.canvases.current().size();

c = mari.canvases.current().pickColor(Size.width() / 2, Size.height() / 2)

print c.toString()

Parameters:
  • X (int) - The x coordinate of the pixel. These coordinates are relative to the top left corner of the canvas
  • Y (int) - The y coordinate of the pixel. These coordinates are relative to the top left corner of the canvas
  • Space ( PickSpace ) - The space in which to pick pixels in
Returns: Color
The color of the pixel at the canvas location (x, y)
Raises:
  • ValueError - Emitted if X or Y are outside of the range (0, 0) - size()
