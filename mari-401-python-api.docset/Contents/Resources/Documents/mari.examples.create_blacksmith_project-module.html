createBlacksmithProject ()

source code  

This creates a project using the BlackSmith assets from the web site.

At the time of writing, the assets for this script can be downloaded from: http://www.thefoundry.co.uk/products/mari/training/#blacksmith

The assets should be extracted from the zip files into a directory, which should end up looking like this:

  • Anvil
  • Apron
  • blacksmith
  • blacksmith_eyes
  • blacksmithObjects
  • Goggles
  • Hammer
  • Tongs

Please ensure the assets are set up correctly before running the script.
