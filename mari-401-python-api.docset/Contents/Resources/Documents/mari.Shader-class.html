            Shader

Provides access to Mari's real-time layer based display shaders.

Shaders are modules that combine the layer system's Channel objects to produce a rendered output.

You can access shaders from a GeoEntity - for example: mari.geo.current().shaderList()

Example Code 

>>> # This example shows how to obtain the current shader
>>> import mari
>>> current_shader = mari.geo.current().currentShader()
>>> print current_shader.className()
Shader
