Class ColorPickerMode

source code

These are used to define the various modes that the color swatches and pickers can be displayed in.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  COLOR_PICKER_RAW = 0
No colorspace conversion is applied.
  COLOR_PICKER_COLOR_MANAGER = 1
The color values are converted using the current color profile set on the color manager.
  COLOR_PICKER_OCIO = 2
The color values are converted to the current default monitor colorspace set on the current project using OCIO.
Class Variable Details

COLOR_PICKER_RAW
