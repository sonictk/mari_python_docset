Class MetadataFlag

source code

Specifies the options that can be set on metadata items.

These options can be combined using binary 'or' operations - for example: 

>>> import mari
>>> channel = mari.geo.current().currentChannel()
>>> channel.setMetadata("MyVersion", "2.1")
>>> channel.setMetadataFlags('MyVersion', channel.METADATA_VISIBLE | channel.METADATA_SAVED)

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  METADATA_VISIBLE = 1
The item is visible and can be seen in the corresponding GUI.
  METADATA_EDITABLE = 2
The item's value can be modified.
  METADATA_REMOVABLE = 4
The item can be removed by users.
