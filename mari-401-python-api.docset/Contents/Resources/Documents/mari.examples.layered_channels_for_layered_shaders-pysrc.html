 10      ''' 
 11      This example class uses the supplied LayerShader's post-shader compositing setup to update the used Channels to have matching pre-shader compositing. 
 12      The layer arragment in the layered shader will be mirrored in each of the input channels. 
 13      Rerunning the script will update as much of the used channels as possible. 
 14   
 15      This is a simple example and has the following limitations: 
 16   
 17        - B{Layer Order} 
 18   
 19          The order of the top level layers in a channel must match the shader order in the layered shader. 
 20          If you reorder the layers in channels things will fail. 
 21   
 22        - B{Shader Order} 
 23   
 24          The order of the top level layers in a channel must match the shader order in the layered shader. 
 25          If you reorder the shader layers in the layered shader things will fail. 
 26          If a new shader layer is required add it to the bottom of the layer stack. 
 27   
 28        - B{Changing Masks} 
 29   
 30          If you change a layer shader's mask from a paintable to an mask stack the channel layers will not be updated. 
 31          To resolve this, manually delete the masks from the channel layers. 
 32   
 33        - B{Convert Paint Layer to Group Layer} 
 34   
 35          The script will use the layers that are already in the channels. 
 36          If a layer is a paintable layer and it has already been assigned a mask and you would like it converted to a group do the following: 
 37            - Create a group in the same place. 
 38            - Drag the paintable layer into the group. 
 39            - Remove the mask from the paitable layer. 
 40            - Rerun the script. 
 41   
 42      To use this script, use the following basic steps: 
 43   
 44        - Create a new project 
 45        - Create a layered shader (click on the plus button on the shaders widget and choose "Layered") 
 46        - Go to the "Shader Layers" tab and create any 3 shaders (click on the plus button and choose from the list) 
 47        - then run this script via Python > Shaders > Update Layered Channels 
 48      '''
