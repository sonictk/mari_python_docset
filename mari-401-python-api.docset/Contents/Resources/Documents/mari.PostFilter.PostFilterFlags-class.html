Class PostFilterFlags

source code

Specifies the options that can be set on filters.

These options can be combined using binary 'or' operations - for example: 

>>> import mari
>>> filter_collection = mari.gl_render.createPostFilterCollection("New Filter Collection")
>>> file_lut = filter_collection.createFileLUT("New File LUT")
>>> file_lut.setFlags(file_lut.POSTFILTER_ENABLED | file_lut.POSTFILTER_REMOVABLE)
>>> mari.gl_render.deletePostFilterCollection(filter_collection)

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  POSTFILTER_ENABLED = 1
Whether the filter is enabled.
  POSTFILTER_REMOVABLE = 2
The filter can be removed by users.
