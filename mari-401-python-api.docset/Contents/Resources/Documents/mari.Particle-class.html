exportGeo ( self , FileName , Object , UVImageSizes , Flags , Frame , GeoVersion = None )

source code  

Exports a predefined set of attributes to a particle file on disk. 

>>> geo_entity = mari.geo.current()
>>> mari.particle.exportGeo('/tmp/filename.bgeo', geo_entity, {0: 256, 10: 512 },
...                         mari.particle.EXPORT_NORMAL, 0)
Parameters:
  • FileName (str) - The name of the file to export to. Any existing file will be overwritten.
  • Object ( GeoEntity ) - The object to export attributes for.
  • UVImageSizes (variant) - A dictionary that maps integer patch indices to sizes to export at. Only patch indices specified in the dictionary will be exported.
  • Flags (int) - The options to export with. See ExportFlags .
  • Frame (int) - The animated frame number to export for.
  • GeoVersion ( GeoEntityVersion ) - The geometry version to export for. If None then the current geometry version is used.
Returns: bool
True if the export completed successfully, or False otherwise.
Raises:
  • TypeError - Raised if the patch sizes dictionary was in the wrong format.
  • ValueError - Raised if no object was given, no patch sizes are given, the patch indices are invalid, the patch sizes are invalid.

exportImageSets ( self , FileName , ImageSets , Flags , Frame , UVIndices , GeoVersion = None )

source code  

Exports a set of image sets to a particle file on disk.

The image sets must belong to the same object and have the same width and height. >>> geo_entity = mari.geo.current() >>> image_set = geo_entity.currentImageSet() >>> mari.particle.exportImageSets('/tmp/filename.bgeo', {'color' : image_set}, mari.particle.EXPORT_ALPHA, ... 0, [0, 1])

Parameters:
  • FileName (str) - The name of the file to export to. Any existing file will be overwritten.
  • ImageSets (variant) - A dictionary that maps string particle attribute names, to export to, to image sets, to export from.
  • Flags (int) - The options to export with. See ExportFlags .
  • Frame (int) - The animated frame number to export for.
  • UVIndices (list of int) - Which patches to export for. Passing in an empty list will cause all patches to be exported.
  • GeoVersion ( GeoEntityVersion ) - The geometry version to export for. If None then the current geometry version is used.
Returns: bool
True if the export completed successfully, or False otherwise.
Raises:
  • ValueError - Raised if no image sets are given, the image sets are invalid, the patch indices are invalid, the patch sizes don't match or they don't belong to the same object.

importImageSets ( self , FileName , ImageSets , Interpolation , Flags , Frame , UVIndices , GeoVersion = None , SearchRange = 1.0 , SearchUnits = 0 , Search = 0 , SampleCount = 16 )

source code  

Imports into a set of image sets from a particle file on disk.

The image sets must belong to the same object and have the same width and height. >>> geo_entity = mari.geo.current() >>> image_set = geo_entity.currentImageSet() >>> mari.particle.importImageSets('/tmp/filename.bgeo', {'color' : image_set}, ... mari.particle.INTERPOLATION_INVERSE_DISTANCE_WEIGHTED, ... mari.particle.IMPORT_BLEED_EDGES, 0, [0, 1])

Parameters:
  • FileName (str) - The name of the file to import to.
  • ImageSets (variant) - A dictionary that maps string particle attribute names, to import from, to image sets, to import into.
  • Interpolation ( InterpolationMethod ) - Which method of interpolation to use when calculating the final color value from the closest particles.
  • Flags (int) - The options to import with. See ImportFlags .
  • Frame (int) - The animated frame number to import to.
  • UVIndices (list of int) - Which patches to import into. Passing in an empty list will cause all patches to be imported.
  • GeoVersion ( GeoEntityVersion ) - The geometry version to import to. If None then the current geometry version is used.
  • SearchRange (float) - The maximum distance allowed to be searched.
  • SearchUnits ( RangeUnits ) - The units of measure for the search range.
  • Search ( SearchMethod ) - The method used to search the particle cloud with from each of destination points.
  • SampleCount (int) - The number of samples to search for and interpolate together to produce the final color.
Returns: bool
True if the import completed successfully, or False otherwise.
Raises:
  • IOError - Raised if the file was invalid; if the file was missing data; if any of the file data was in the wrong format.
  • TypeError - Raised if the image set dictionary was in the wrong format.
  • ValueError - Raised if no image sets were supplied; if any of the image sets was invalid; if the image sets belong to different objects; if any of the patch indices was invalid; if any of the patch sizes didn't match between image sets.

transferChannel ( self , SourceChannel , SourceFrame , SourceUVIndices , DestinationObject , DestinationFrame , DestinationUVIndices , Interpolation , DestinationFlags , SourceGeoVersion = None , DestinationGeoVersion = None , SearchRange = 1.0 , SearchUnits = 0 , FillColor = None , Search = 0 , SampleCount = 16 , SourceFlags = 32768 , DestinationSize = 0 )

source code  

Copies a single channel from one object to another. 

>>> src_geo_entity = mari.geo.get('Head1')
>>> src_channel = src_geo_entity.channel('diffuse')
>>> dst_geo_entity = mari.geo.get('Head2')
>>> mari.particle.transferChannel(src_channel, 0, [], dst_geo_entity, 0, [],
...                               mari.particle.INTERPOLATION_INVERSE_DISTANCE_WEIGHTED,
...                               mari.particle.IMPORT_BLEED_EDGES | mari.particle.IMPORT_FLATTEN)
Parameters:
  • SourceChannel ( Channel ) - The source channel to read from.
  • SourceFrame (int) - The animated frame number to read from.
  • SourceUVIndices (list of int) - Which patches in the source to read from. Passing in an empty list will cause all source patches to be read from.
  • DestinationObject ( GeoEntity ) - The destination object to write into.
  • DestinationFrame (int) - The animated frame number to write to.
  • DestinationUVIndices (list of int) - Which patches in the destination to write to. Passing in an empty list will cause all destination patches to be written to.
  • Interpolation ( InterpolationMethod ) - Which method of interpolation to use when calculating the final color value from the closest particles.
  • DestinationFlags (int) - The options used in writing. See ImportFlags .
  • SourceGeoVersion ( GeoEntityVersion ) - The source geometry version to read from. If None then the current geometry version is used.
  • DestinationGeoVersion ( GeoEntityVersion ) - The destination geometry version to write into. If None then the current geometry version is used.
  • SearchRange (float) - The maximum distance allowed to be searched.
  • SearchUnits ( RangeUnits ) - The units of measure for the search range.
  • FillColor ( Color ) - The color used to fill the newly created destination image sets with.
  • Search ( SearchMethod ) - The method used to search the particle cloud with from each of destination points.
  • SampleCount (int) - The number of samples to search for and interpolate together to produce the final color.
  • SourceFlags (int) - The options used in reading. See ExportFlags .
  • DestinationSize ( ImageSet.Size ) - Used to specified the image size for all uv indices in the destination channel. If a size is not given then the size will match the source.
Returns: Channel
A destination channel if the transfer completed successfully, or None otherwise.
Raises:
  • ValueError - Raised if no channel was supplied; if any of the patch indices was invalid; if no destination object was supplied.

source code  

Copies image data from one image set to another.

The image sets in the source list must belong to the same object and have the same width and height. This also applies to the destination list. >>> src_geo_entity = mari.geo.get('Head1') >>> src_image_set = src_geo_entity.currentImageSet() >>> dst_geo_entity = mari.geo.get('Head2') >>> dst_image_set = dst_geo_entity.currentImageSet() >>> mari.particle.transferImageSets([src_image_set], 0, [0, 1], [dst_image_set], 0, [0, 1], ... mari.particle.INTERPOLATION_INVERSE_DISTANCE_WEIGHTED, ... mari.particle.IMPORT_BLEED_EDGES)

Parameters:
  • SourceImageSets (list of ImageSet ) - A list of source image sets to read from.
  • SourceFrame (int) - The animated frame number to read from.
  • SourceUVIndices (list of int) - Which patches in the source to read from. Passing in an empty list will cause all source patches to be read from.
  • DestinationImageSets (list of ImageSet ) - A list of destination image sets to write into.
  • DestinationFrame (int) - The animated frame number to write to.
  • DestinationUVIndices (list of int) - Which patches in the destination to write to. Passing in an empty list will cause all destination patches to be written to.
  • Interpolation ( InterpolationMethod ) - Which method of interpolation to use when calculating the final color value from the closest particles.
  • DestinationFlags (int) - The options used in writing. See ImportFlags .
  • SourceGeoVersion ( GeoEntityVersion ) - The source geometry version to read from. If None then the current geometry version is used.
  • DestinationGeoVersion ( GeoEntityVersion ) - The destination geometry version to write into. If None then the current geometry version is used.
  • SearchRange (float) - The maximum distance allowed to be searched.
  • SearchUnits ( RangeUnits ) - The units of measure for the search range.
  • Search ( SearchMethod ) - The method used to search the particle cloud with from each of destination points.
  • SampleCount (int) - The number of samples to search for and interpolate together to produce the final color.
  • SourceFlags (int) - The options used in reading. See ExportFlags .
Returns: bool
True if the transfer completed successfully, or False otherwise.
Raises:
  • ValueError - Raised if no image sets were supplied; the number of source image sets doesn't match the number of destination image sets; if any of the image sets was invalid; if the image sets belong to different objects; if any of the patch indices was invalid; if any of the patch sizes didn't match between image sets.

transferLayers ( self , SourceLayers , SourceFrame , SourceUVIndices , DestinationStack , DestinationFrame , DestinationUVIndices , Interpolation , DestinationFlags , SourceGeoVersion = None , DestinationGeoVersion = None , SearchRange = 1.0 , SearchUnits = 0 , FillColor = None , Search = 0 , SampleCount = 16 , SourceFlags = 32768 )

Copies layers from one object to another.

The layers in the source list must belong to the same object. >>> src_geo_entity = mari.geo.get('Head1') >>> src_channel = src_geo_entity.channel('diffuse') >>> src_layer = src_channel.layer('color') >>> dst_geo_entity = mari.geo.get('Head2') >>> dst_channel = dst_geo_entity.channel('diffuse') >>> mari.particle.transferLayers([src_layer], 0, [], dst_channel, 0, [], ... mari.particle.INTERPOLATION_INVERSE_DISTANCE_WEIGHTED, ... mari.particle.IMPORT_BLEED_EDGES | mari.particle.IMPORT_FLATTEN)

Parameters:
  • SourceLayers (list of Layer ) - A list of source layers to read from.
  • SourceFrame (int) - The animated frame number to read from.
  • SourceUVIndices (list of int) - Which patches in the source to read from. Passing in an empty list will cause all source patches to be read from.
  • DestinationStack ( LayerStack ) - The destination layer stack to write into.
  • DestinationFrame (int) - The animated frame number to write to.
  • DestinationUVIndices (list of int) - Which patches in the destination to write to. Passing in an empty list will cause all destination patches to be written to.
  • Interpolation ( InterpolationMethod ) - Which method of interpolation to use when calculating the final color value from the closest particles.
  • DestinationFlags (int) - The options used in writing. See ImportFlags .
  • SourceGeoVersion ( GeoEntityVersion ) - The source geometry version to read from. If None then the current geometry version is used.
  • DestinationGeoVersion ( GeoEntityVersion ) - The destination geometry version to write into. If None then the current geometry version is used.
  • SearchRange (float) - The maximum distance allowed to be searched.
  • SearchUnits ( RangeUnits ) - The units of measure for the search range.
  • FillColor ( Color ) - The color used to fill the newly created destination image sets with.
  • Search ( SearchMethod ) - The method used to search the particle cloud with from each of destination points.
  • SampleCount (int) - The number of samples to search for and interpolate together to produce the final color.
  • SourceFlags (int) - The options used in reading. See ExportFlags .
Returns: list of Layer
A list of destination layers if the transfer completed successfully, or an empty list otherwise.
Raises:
  • ValueError - Raised if no layers were supplied; if any of the layers was invalid; if the layers belong to different objects; if any of the patch indices was invalid; if no destination layer stack was supplied.
