Class ExportFlags

source code

Specifies options for use in exporting.

Note: These values are exposed in the parent class, but are also documented here for convenience.

Class Variables
  EXPORT_CENTER_POSITIONS = 1
Subtracts the object's center position from each of the particle positions.
  EXPORT_GEOMETRY_ID = 2
Include attribute to identify which object the particle belongs to (name = geometryId, type = indexed string).
  EXPORT_PATCH_ID = 4
Include attribute to identify which patch the particle belongs to (name = patchId, type = int).
  EXPORT_PARTICLE_ID = 8
Include attribute to uniquely identify the particle (name = id, type = int).
  EXPORT_PIXEL_COORDINATE = 16
Include attribute to identify the pixel within the patch that the particle belongs to (name = pixelCoordinate, type = int[2]).
  EXPORT_UV = 32
Include attribute for the object's texture coordinates at the particle position (name = uv, type = float[2]).
  EXPORT_NORMAL = 64
Include attribute for the object's world-space normal at the particle position (name = normal, type = vector).
  EXPORT_TANGENT = 128
Include attribute for the object's world-space tangent at the particle position (name = tangent, type = vector).
  EXPORT_BITANGENT = 256
Include attribute for the object's world-space bitangent at the particle position (name = bitangent, type = vector).
  EXPORT_ALPHA = 512
Include alpha channel of all image set attributes.
  EXPORT_AMBIENT_OCCLUSION = 1024
Include ambient occlusion attribute at the particle position (name = ambientOcclusion, type = float).
  EXPORT_IGNORE_MEMORY_REQUIREMENT = 2048
The operation will still continue even if the system doesn't have enough memory available.
  EXPORT_RADIUS = 4096
Include attribute for the radius that encompasses the particle's immediate neighbours (name = radius, type = float).
  EXPORT_IGNORE_HIDDEN = 8192
The operation ignores hidden parts of the geometry.
  EXPORT_IGNORE_UNSELECTED = 16384
The operation ignores unselected parts of the geometry.
  EXPORT_MATCH_IMPORT = 32768
This is a special option only used in the transfer functions that makes the export options match the given import options.
