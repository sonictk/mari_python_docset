Class UpdateChannelLayers

source code

This example class uses the supplied LayerShader's post-shader compositing setup to update the used Channels to have matching pre-shader compositing. The layer arragment in the layered shader will be mirrored in each of the input channels. Rerunning the script will update as much of the used channels as possible.

This is a simple example and has the following limitations:

  • Layer Order

    The order of the top level layers in a channel must match the shader order in the layered shader. If you reorder the layers in channels things will fail.

  • Shader Order

    The order of the top level layers in a channel must match the shader order in the layered shader. If you reorder the shader layers in the layered shader things will fail. If a new shader layer is required add it to the bottom of the layer stack.

  • Changing Masks

    If you change a layer shader's mask from a paintable to an mask stack the channel layers will not be updated. To resolve this, manually delete the masks from the channel layers.

  • Convert Paint Layer to Group Layer

    The script will use the layers that are already in the channels. If a layer is a paintable layer and it has already been assigned a mask and you would like it converted to a group do the following:

    • Create a group in the same place.
    • Drag the paintable layer into the group.
    • Remove the mask from the paitable layer.
    • Rerun the script.

To use this script, use the following basic steps:

  • Create a new project
  • Create a layered shader (click on the plus button on the shaders widget and choose "Layered")
  • Go to the "Shader Layers" tab and create any 3 shaders (click on the plus button and choose from the list)
  • then run this script via Python > Shaders > Update Layered Channels
